# TMC Kiosk

### Prerequisites

Xubuntu 24.04

### Installation
Firstly open the terminal with
```
SUPER + T
```
Then clone this repository with
```
git clone https://github.com/caleb-griffiths/tmc-kiosk.git
```
Then enter the directory with
```
cd tmc-kiosk
```
Then run the setup with
```
./setup.sh
```
Once the setup has finished, reboot the system with
```
sudo reboot
```
